# Search app
We use Algolia to index our content and search it.

So, you can search for anything you want.

# CLI
```
horsy search <query>
```
Query can be part of app name or description.

# TUI
```
horsy
6
<query>
```

# GUI
Open `Browse` tab

In `Search packages` type your query and press `Search` button
